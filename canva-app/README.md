# ContentLab — Canva App

## Setup (first time only)
```bash
cd canva-app
npm install
```

## Run
```bash
npm start
```
Then open Canva Developer Portal and preview the app.

## What it does
- User designs a graphic in Canva
- Clicks "Save Design to ContentLab"  
- Design exports as PNG → uploads to ContentLab media library
- User goes to ContentLab → attaches it to a blog post or social post
